## 🚀 如何使用 (Usage)

1. **安装管理器**：首先需要在浏览器中安装脚本管理器扩展，推荐 **[Tampermonkey](https://www.tampermonkey.net/)** (篡改猴)。
2. **选择脚本**：点击上方列表中的项目链接进入对应目录。
3. **安装脚本**：在目录中找到以 `.user.js` 结尾的文件，点击文件内容页面的 "Raw" 按钮（或直接复制内容）即可触发安装。
